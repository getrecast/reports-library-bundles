 # Recast Report Template Bundles

  Public download repo for Recast report template bundles.

  Each ZIP contains a self-contained report template bundle with:

  - the report template
  - a Python build script
  - setup instructions
  - API usage notes
  - agent-facing guidance for rebuilding the report with your own Recast data

  ## Available Bundles

  - `channel-mroi-bundle.zip`
  - `incrementality-cpa-bundle.zip`
  - `period-comparison-bundle.zip`
  - `saturation-curves-bundle.zip`
  - `spend-decomposition-bundle.zip`
  - `testing-roadmap-bundle.zip`
  - `whitelabel-insights-bundle.zip`

  ## How To Use

  1. Download the ZIP for the report you want.
  2. Unzip it locally.
  3. Follow the included `README.md` and `INSTALL.md`.
  4. Add your Recast API token to the local `.env` file.
  5. Run the bundle against your own Recast client and KPI.

  ## Notes

  These bundles are distributed here so the download links work for external users. The main Recast research pages remain at:

  https://research.getrecast.com/report-template-library/
